linux 基础

学习Linux基础的笔记，视频教程来源慕课网：Linux达人养成计划II

1. shell基础

1.1 shell基础

shell是一个命令行解释器，与内核交互
linux 使用Bash 可以使用echo $SHELL 来查看shell类型

1.2 VI编辑器

Visual Interface 编辑器
VI编辑器的操作模式

  • command mode:命令模式
    h 光标左移
    j 光标下移
    k 光标上移
    l 光标右移
    ctrl+f 向下翻页(front)
    ctrl+b 向下翻页(back)
    ctrl+d 向下翻半页(down)
    ctrl+u 向上翻半页(up)
    dd 删除光标所在行
    o 在光标所在行的下方插入一行并切换到输入模式
    yy 复执光标所在行
    p 在光标所在行的下方粘贴
    P 在光标所在行的上方粘贴

  • insert mode:输入模式

  • last line mode:底行模式也可归于命令模式
    :w 保存
    :q 退出
    :! 强制
    :ls 列出所有文件
    :n 打开后一个文件 :N 打开前一个文件
    :15 把光标快速定义到15行
    /xxx 从光标位置向后搜索字符xxx
    ?xxx 从光标位置向前搜字符xxx

    1.2 脚本执行方式

    echo [选项] [内容]
    -e 可以输入转义字符的内容
    脚本执行
  1. 赋予执行权限，直接运行
    chmod 755 hello.sh
    ./hell.sh
  2. 通过Bash调用执行脚本
    bash hello.sh

    1.3 Bash的基本功能

    1.3.1 命令别名与快捷键

    alias 查看系统中所有命令的别名
    alias ls=’ls —color==never’ 修改ls的命令别名
    alias 别名=‘原命令’ 设定命令别名
    别名永久生效与删除别名
    vi ~/.bashrc 写入环境变量配置文件
    unalias 别名 删除别名
    快捷键
    ctrl+l 清屏 clear
    ctrl+c 强制终止当前命令
    ctrl+a 光标移动到命令行首
    ctrl+e 光标移动到命令行尾
    ctrl+u 从光标所在位置删除到行首
    ctrl+z 把命令放入后台
    ctrl+r 在历史命令中搜索

    1.4 输出重定向

  3. 标准输入输出
    | 设备 | 设备文件名 | 文件描述符 | 类型 |
    | ——— | ——-: | :——: | :——- |
    | 键盘 | /dev/stdin | 0 | 标准输入 |
    | 显示器 | /dev/stdout| 1 | 标准输出 |
    | 显示器 | /dev/stderr| 2 | 标准错误输出 |

  4. 输出重定向

    > 标准输出重定向，单个大于号是覆盖
    >> 标准输出重定向，双大于号是追加
    2> 标准错误重定向，覆盖
    2>> 标准错误重定向，追加
    命令 > 文件 2>&1 已覆盖方式，标准错误和标准输出重定向到文件中
    命令 >> 文件 2>&1 以追加方式，
    命令 &> 文件 以覆盖方式，标准错误和标准输出重定向到文件中
    命令 &>> 文件 以追加方式，
    命令 >> 文件1 2>>文件2 把标准输出追加到文件1中，把标准错误追加到文件2中

  5. 输入重定向 < 小于号
    wc 统计行数，单词数，字符数
    -c 统计字节数
    -w 统计单词数
    -l 统计行数
    命令 < 文件 把文件作为命令的输入
    命令 << 标识符 双小于号 极少用

    1.5 管道符

    多命令顺序执行

  6. 命令1; 命令2 命令1成功与否，命令2顺序执行
  7. 命令1&&命令2 逻辑与，命令1成功，命令2才执行，否则不执行
  8. 命令1||命令2 逻辑或，命令1成功，命令2不执行，否则执行

    管道符

    命令1 | 命令2 命令1的正确输出作为命令2的操作对象
    netstat -an | grep “ESTABLISHED” 查看连接到服务器的情况

    通配符

  • ? 匹配一个任意字符
    • 匹配任意个字符
  • [] 匹配括号中任意一个字符
  • [-] 匹配括号中任意一个字符，-代表范围
  • [^] 逻辑非，表示匹配不是中括号内的一个字符

    2. 磁盘管理

    MBR分区
  1. Linux系统中硬件设备都是以文件的形式存在于根目录下的dev目录下
  2. 必须对磁盘进行分区，格式化，挂载后才能使用
    fdisk

    3. 用户和用户组管理

    /etc/group存储当前系统种所有用户组信息
    group:x:0:yangz
    组名：组密码：组编号：组中用户列表
    /etc/gshadow存储当前系统种用户组的密码
    group:: : :
    组名称：组密码：组管理者：组中用户名列表
    /etc/passwd
    user:x:123:456: ： ：
    用户名：密码占位符：用户编号：用户组编号：用户注释信息：用户主目录：shell类型
    */etc/shadow

    3.1 用户和用户组基本命令

    groupadd 【用户组名】添加用户组
    groupmod -n 【新用户组名】【旧用户组名】 修改用户组名
    groupmod -g 【用户组编号】【用户组名】修改用户组编号
    groupdel [用户组名]s 删除用户组，需要先删除用户才能删除用户组

useradd -g [用户组名]【用户名】 添加用户到用户组
useradd -d /home/xxx imooc 指定用户个人文件夹
usermod -g [用户组名] 【用户名】 将用户添加到用户组中
userdel -r [用户名] 连同用户文件一起删除

passwd -l [用户名] 锁定一个用户的登陆
passwd -u [用户名] 解锁一个用户的登陆
passwd -d [用户名] 删除用户密码，可以无密码登陆

3.2 用户和用户组进阶命令

gpasswd -a [用户名] [用户组]将用户添加到某个附属组
newgrp [附属组名称] 临时切换用户组到附属组
gpasswd -d [用户名] [用户组]将用户从某个附属组中删除
useradd -g [主用户组] -G [附属用户组。。。。] 添加用户时指定用户组
gpasswd [用户组名] 修改用户组密码
whoami 我是谁?
id [用户名] 显示用户详细信息
groups 【用户名】 显示用户所在组
chfn [用户名] 设置用户详细资料
finger [用户名] 显示用户详细资料
