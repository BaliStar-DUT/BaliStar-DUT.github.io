Linux C语言编程基本原理与实践

第一节 C语言历史

ANSI C ：
二十世纪年代由美国国家标准局制定的C语言国际标准语法：ANSI C。
C语言用途：
linux嵌入式，ARM嵌入式，单片机，硬件编程
NGINX：C语言编写的高并发web服务器

第二节 开发环境与配置

linux和cc(gcc)编译器
emacs和vim编辑器
常用命令：
sudo apt-get update //自动更新资源
sudo apt-get install vim; //安装vim编辑器
cc -v 查看编译器版本
vim常用操作：
i:左插入 shift+i:行首插入
a:右插入 shift+a:行尾插入
o:向下插入一行 shift+o:向上插入一行
x:删除当前字符 dd:删除当前行

第三节 编写C程序

头文件：
导入库文件：#include //导入标准输入输出库，存在于linux /usr/include里.尖括号里面的是系统函数库里的，会在环境变量里查找文件，双引号是当前目录里查找。include 表示编译时直接复制过来。

#include<stdio.h>
int main()
{
    printf("Hello World!\n");
    return 0;
}

编译程序：
cc a.c

第四节 多文件编译

4.1 编写多文件程序

gcc a.c b.c -o outputfile //编译并连接成可执行文件,可以仅连接
gcc -c a.c -o a.o //仅编译但不连接,可以通过这种方法将大型软件工程的源文件先编译成静态库，可以直接调用。

//a.c 与b.c在同一文件夹下
vim中同时打开多个文件：
:set nu;// 显示行号
:sp b.c //同时打开a.c和b.c
ctrl+w+↓ //进入下面的文件
ctrl+w+↑ //进入上面的文件
:dd //复制
:p //粘贴
:wqa //保存所有文件

#include<stdio.h>
int max(int a,int b)
{
    if(a>b){
        return a;
    }else{
        return b;
    }
}
int main()
{
    int a1=2;
    int a2=3;
    int maxNum;
    maxNum=max(a1,a2);
    printf("the max value is %d",maxNum);
    return 0;
}

4.2 头文件与函数分离

为了使别人调用函数时清楚知道函数返回值，参数类型，可以将函数的声明和实现分离，声明写成.h头文件的形式，这样调用时只要将头文件包含进来就可以了。

4.3 makefile编写

make工具可以将大型的开发项目分成分成若干个模块。
make内部也是使用gcc.

make -v //显示make版本号
#this is make file
hello.out:max.o min.o a.c
    gcc max.o min.o a.c
max.o:max.c
    gcc -c max.c
min.o:min.c
    gcc -c min.c

第五节 main函数详解

5.1 return语句

前一个语句成功时，才运行后一个语句。
gcc main.c -o main.out && ./main.out
每个语句都有一个返回值。
return 0; 表示程序运行成功

5.2 main函数参数

main(int argc,char *argv[]) //argc 表示参数个数，argv[]表示参数值；

#include <stdio.h>
int main(int argv,char* argc[])
{
    printf("argv is %d \n",argv);
    int i=0;
    for(;i<argv;i++){
        printf("argc[%d] is %s",i,argc[i]);
    }
    return 0;
}

6.1输入输出函数

stdin //标准输入，键盘
stdout //标准输出，1，显示器
stderr //标准错误,2

fprintf(stdout,”please input a num:\n”);
scanf(“%d”,&a);
fscanf(stdin,”%d”,&a);

 #include <stdio.h>
int main(int argv,char* argc[])
{

    if(a<0){

        return 1;    
    }
    //scanf("%d",&a);
    printf("you have input:%d\n",a);
    return 0;
}
