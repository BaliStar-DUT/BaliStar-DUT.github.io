Mysql简单操作

Dec 23, 2015 in Mysql

1. 启动Mysql

(1) 将mysql命令加入到环境变量PATH中
PATH=\$PATH:/usr/local/mysql/bin
(2) 使Mysql开机加载
echo “PATH=$PATH:/usr/local/mysql/bin” >> /etc/profile
source /etc/profile
mysql -u root -p

2. 连接数据库

mysql -uroot -p -h192.168.137.10 -P3306

3. 数据库基本操作

  1. show databases;//列出当前数据库
  2. use mysql; show tables; //选择数据库，列出数据表
  3. desc slow_log;//查看某个表的全部字段
  4. show create table show_log\G; //显示建表语句
  5. select user(); //显示当前是哪个用户
  6. select database();//显示当前是那个数据库
  7. create database db1;//创建数据库
  8. create table t1(‘id’ int(4),’name’ char(40));//创建数据表
  9. select version();//查看当前数据库版本
  10. show status; //查看当前mysql状态
  11. show variables;//查看mysql参数
  12. set global max_connect_errors =1000; //临时修改参数，重启后失效
  13. show processlist;//查看当前mysql服务器队列
  14. grant all on . to user1 identified by ‘123456’;//all表示所有权限，*表示所有数据库数据表，identified by 后面跟密码，这里user1是localhost上的user1
  15. grant all on db1.* to ‘user2’@’10.0.2.100’ identified by ‘111111’; //授权网络上某台主机的用户，ip可以用％代替，表示所有主机
    1. select count(*) from mysql.user; //查询语句
  16. insert into db1.t1 values(1,’abc’); //插入一行
  17. update db1.t1 set name=’aaa’ where id=1; //更新操作
  18. truncate table db1.t1;//清空数据表
  19. drop table db1.t1;//删除数据表
  20. drop database db1;//删除数据库

    4. mysqlmysql数据库的备份与恢复

  21. mysqldump -uroot -p’yourpassword’ mysql >/tmp/mysql.sql //备份数据库mysql到文件mysql.sql
  22. mysql -uroot -p’yourpassword’ mysql </tmp/mysql.sql //从文件中恢复到数据库
