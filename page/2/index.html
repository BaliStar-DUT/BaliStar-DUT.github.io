linux常用命令

学习Linux基础的笔记，视频教程来源慕课网：Linux达人养成计划I

第一节 linux命令格式

1.1 linux文件类型

-rwxrwxrwx 1 root root 1207

  1. 第一位表示文件类型
    • l表示软连接文件
    • d表示目录文件
    • 块设备文件
    • 字符设备文件
    • 套接字文件
    • 管道文件
  2. 左三位表示所有者权限
  3. 中三位表示所在用户组权限
  4. 右三位表示其他用户权限
    ls -i 查看文件id号

    1.2 目录处理命令

  5. 建立目录
    mkdir -p dir1/dir2 递归创建目录
  6. 切换所在目录
    cd dir1
    cd ~ 回到home目录
    cd - 回到上次的目录
    ctrl+l 清理屏幕
  7. 显示当前所在目录
    pwd
    list
    ll = list -l
  8. 删除目录（极少用了）
    rmdir dir1 删除空白目录
  9. 删除文件或目录
    rm -r dir1 删除目录
    rm file1 删除文件
    rm -rf dir1或file1 强制删除，不可恢复
    rm -rf /* linux自杀命令
  10. 复制
    cp -r dir1 dir2 复制目录
    cp -p file1 file2 连带文件属性一起复制
    cp -a dir1 dir2 目标文件与源文件属性一致 相当于cp -pdr
  11. 剪切或改名
    mv dir1或file1 dir2或file2 不同目录下为剪切
    mv file1 file2 同一目录下为改名
  12. 链接命令
    ln -s file1 file2 创建软链接
    硬链接：拥有相同的文件ID,拥有相同的i节点和存储block块，可以看做一个文件。可通过i节点识别，不可跨分区，不能针对目录使用。
    有两个硬链接的文件，删掉一个，另一个还可访问。
    软链接：类似windows的快捷方式。
    软链接有自己的i节点和blocks
    删除原文件，软链接就不能使用了
    软链接中源文件一定要写绝对路径s

    1.3 常用目录作用

  • / 根目录
  • /bin 保存系统命令，所有用户都可用
  • /sbin 保存系统命令，只有超级用户才可用
  • /usr/bin 系统命令，普通用户
  • /usr/sbin 系统命令，超级用户
  • /boot 系统启动目录
  • /dev 特殊文件保存目录
  • /etc 默认配置文件保存目录
  • /home 普通用户文件保存
  • /root 超级用户文件
  • /lib 函数库保存位置
  • /misc,/media,/mnt 空目录，作为外接介质分配盘符
  • /proc,/sys 不能直接操作，保存内存的挂载点，内存的盘符
  • /tmp 临时目录，可以随意放东西
  • /usr 系统软件资源目录
  • /var 系统相关文档内容

    1.4 文件搜索命令

  1. 文件搜索命令 locate 只能搜索文件名
    locate file1 搜索数据库/var/lib/mlocate中记录的文件，搜索速度较快，但是不能找到最新建立的文件,只能搜索文件名。
    updatedb 更新数据库
    命令遵循配置文件/etc/updatedb.conf的规则进行搜索。

  2. 文件搜索命令 find
    find 搜索范围 搜索条件
    find / -name filename 在/下按文件名搜索filename
    find /root -name “ab[cd]” 在/root下按文件名搜索abc或abd
    find /root -name “[cd]” 搜索以c或d结尾的文件
    *通配符    :

    • * 匹配任意内容
    • ？ 匹配任意一个字符
    • [] 匹配任意一个中括号内的字符
      -i 不区分大小写
      -user username 按用户名搜索
      -nouser 按没有用户名的文件搜索，就是搜索垃圾文件或内核产生的文件，或外来文件
      find /root -mtime +10 查找10天前修改的文件
      find /root -mtime -10 查找10天内s修改的文件
      -atime 文件访问时间
      -ctime 改变文件属性的时间
      -mtime 修改文件内容的时间
      find . -size -25k查找当前目录下文件大小小于25k的文件
      find . -size +2M查找当前目录下文件大小大于2M的文件，必须大写
      find . -inum 261211查找当前目录下文件ID为.。的文件
      find /etc -size +20k -a -size -30k 查找目录下大于20k和小于30k大小的文件
      -a 逻辑和
      -o 逻辑或
      find /etc -size +20k -a -size -30k -exec ls -lh {}\;
      -exec 参数后面跟的是command命令，它的终止是以;为结束标志的，所以这句命令后面的分号是不可缺少的，考虑到各个系统中分号会有不同的意义，所以前面加反斜杠。
      {} 花括号代表前面find查找出来的文件名。

  3. 命令搜索命令
    whereis 不能搜索文件，只能搜索系统命令
    whereis cmd 搜索命令cmd所在位置和帮助文档所在位置
    -b 只查找可执行文件
    -m 只查找帮助文档
    which 查看命令和命令别名所在位置，但是不一定都能找到
    cd是shell的内置命令，不是linux内置的命令，故无法在linux系统内找到。

  4. 字符串搜索命令 grep [选项] 字符串 文件名在文件种匹配符合条件的字符串
    grep -i 忽略大小写
    grep -v 排除指定字符串
  5. find命令与grep命令的区别
    find 在系统当中搜索符合条件的文件名，如果需要匹配，使用通配符匹配，是完全匹配
    grep 在文件中搜索符合条件的字符串，如果需要匹配，使用正则表达式进行匹配，是包含匹配，只要包含就能找到

    1.5 帮助命令

  6. man 内容 获取制定内容的帮助
    man -f 内容 == whatis 内容 查看内容所在man的位置
    man -数字 内容 查看man的第几部分的内容
    man -k 内容 == apropos 内容 找到全部包含内容关键字的内容

  7. 其他帮助命令
    命令 —help 获取命令选项帮助信息
    help shell内部命令帮助
    info 内容 找出内容所有的详细的信息

    1.6 压缩与解压缩命令

    常见压缩格式.zip,.gz,.bz2,.tar.gz,.tar.bz2

  8. .zip格式
    zip 压缩文件名 源文件名 压缩文件
    zip -r 压缩文件名 源目录 压缩目录
    unzip 压缩文件 解压缩.zip文件

  9. .gz格式压缩
    gzip 源文件 压缩为.gz格式的压缩文件，源文件会消失
    gzip -c 源文件 > 压缩文件 保留源文件，压缩到其他文件
    gzip -r 目录 压缩目录下所有的子文件，但是不压缩目录
    gzip -d 压缩文件 解压缩文件
    gunzip 压缩文件 解压缩文件

  10. .bz2格式压缩
    bzip2 源文件 压缩为bz2格式，不保留源文件
    bzip2 -k 源文件 压缩后保留源文件
    bzip2 命令不能塔索目录
    bzip2 -d 压缩文件 解压缩，-k保留压缩文件
    bunzip2 压缩文件 解压缩，-k保留压缩文件
  11. .tar打包命令
    tar -cvf 打包文件名 源文件
    -c 打包 -v 显示过程 -f 显示打包后的文件名
    tar -cvf abc.tar abc 将abc文件打包成abc.tar
    tar -xvf 打包文件名 解打包文件
    tar -zcvf 压缩包名.tar.gz 源文件 直接将文件压缩成.tar.gz格式
    tar -zxvf 压缩包名.tar.gz 解压缩
    tar -jcvf 压缩包名.tar.bz2 源文件 直接将文件压缩成.tar.bz2格式
    tar -jxvf 压缩包名.tar.bz2 解压缩
    tar -jxvf fiel.tar.bz2 -C /dir 制定解压目录

    1.7 关机或重启命令

  12. shutdown [选项] 时间 保存相关内容，更安全
    -c: 取消前一个关机命令
    -h: 关机
    -r: 重启
    以下命令关机不太安全，推荐用shutdown
    halt
    poweroff
    init 0
    reboot
    init 6
    系统运行级别：0-6
    cat /etc/inittab 修改系统默认运行级别
    runlevel 查询系统运行级别

    1.8 挂载命令

  13. mount [-t 文件类型] [-o 特殊选项] 设备文件名 挂载点 挂载文件系统
    -t 文件系统
    mount 查看系统中已经挂载好的设备
    mount -a 依据配置文件/etc/fstab的内容，自动挂载
  14. umount 设备文件名或挂载点 卸载文件系统
    umount /home/yangz/UUI 卸载文件系统
    fdisk -l：查看U盘设备文件名
    mount -t vfat /dev/sdb1 /mnt/usb 挂载U盘
    linux默认不支持NTFS设备

    1.9 用户登陆与交互命令

  15. w 查看当前用户登陆信息
  16. last 查看历史登陆信息
  17. lastlog 查看所有用户的最后一次登陆时间

linux网络管理

学习Linux网络管理的笔记，视频教程来源慕课网：Linux网络管理

第一节

1.1 ISO/OSI 七层模型

应用层—>表示层—>会话层—>传输层—>网络层—>数据连接层—>物理层
物理层：设备之间的比特流的传输、物理接口、电气特性等;
数据链路层：成帧（写入mac地址）、用MAC地址访问媒介，错误检测与修正
网络层：提供逻辑地址、选路（自动选路）
传输层：可靠与不可靠的传输、传输前的错误检测，流量控制。确定端口号，两种协议TCP、UDP
会话层：对应用会话的管理、同步
表示层：数据的表现形式、特定功能的实现
应用层：用户接口

1.2 TCP/IP

网络接口层：负责监视数据在主机和网络之间的交换
arp协议（地址解析协议）：将IP地址转换成物理地址，局域网内通过MAC地址传输，交换机只识别物理地址
cmd中 arp -a查看与本机有联系的IP地址和mac地址映射
网际互联层：解决主机到主机的通信问题
IP协议：
ICMP协议：PING命令的原理
传输层：
TCP协议：三次握手，面向连接可靠的协议 两军问题
UDP协议：
应用层：
数据封装过程：
FTP默认端口：21

1.3 IP地址

IP包头：
IP地址分类：
A类地址：1.0.0.0—126.255.255.255
B类地址：128.0.0.0—191.255.255.255
C类地址：192.0.0.0—223.255.255.255
D类和E类地址不对民用开放
用标准子网掩码来分割地址。
每个网段内有私有IP地址，用来局域网内分配。
私有IP不能直接访问公网，免费可以自己分配。
192.168.0.0—192.168.255.255

1.4 子网掩码

用来分割网络IP和主机IP
标准子网掩码和变长子网掩码。
IP和子网掩码同时使用才能确认此IP，不是同一网段需要用路由器路由。

1.5 端口的作用

端口是什么：传输层加入的，通过IP确定主机，通过端口确定服务类型。
www服务端口号：80
TCP协议包头和UDP协议包头中有端口号。
端口号长度是16位，一万以内的端口号给系统用的，一万以外的给自己用。
FTP（文件传输协议）：端口号 20 21
SSH （安全shell协议）：端口号 22
telnet（远程登录协议）：23
DNS（域名系统）：53
http：80
SMTP：25 发信
POP3:110 收信
查看本机启用的端口：
命令 ： netstat -an
UDP是无连接的，无状态的。

1.6 DNS的作用

早期hosts文件是做静态IP和域名对应，优先于DNS来解析IP地址。
但是名称解析效能下降和主机维护困难

DNS服务
层次性和分布性
首先向DNS服务器发送一个IP解析请求，客户机拿到IP后访问web服务器。

域名空间结构：

  • “.”根域名，根域名服务器全球只有13台，所有DNS服务器都知道根在哪儿。
  • 顶级域名：ISO分配的，固定的
  • 二级域名：个人或企业向服务商买的
  • 主机名：三级域名是二级域名购买后自己规定的。

如何防范钓鱼网站？

DNS查询过程:
客户机查询本地域名服务器，如果本地域名服务器没有，询问根DNS服务器，根知道顶级域名DNS在哪儿，查询顶级域名DNS服务器，接着查询二级域名服务器。
迭代查询：根服务器，返回优化的查询结果，允许返回优化的值。返回的不一定是准确结果。
递归查询：本地域名服务器，一定要返回一个结果。要么成功，要么失败，一定要有结果。一般客户机与服务器之间的是递归查询。

1.7 网关作用

网关又称网间连接器，协议转换器 ，是一个服务器或路由器
MAC地址，48位。交换机闪烁时，记录插入端口的网卡的地址。交换机不认识IP，属于数据链路层的，IP属于网络层的。

  1. 局域网之间通信不需要，不同网段通信需要通过网关。
  2. 内网通过网关变成公网IP，公网IP转换成内网IP

第三节 Linux网络命令

3.1 网络环境查看命令

  • ifconfig：查看与配置网络状态
  • ifdown:ifdown 网卡设备名 禁用该网卡设备
