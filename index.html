Mysql常见问题

Dec 18, 2015 in Mysql

1. MYSQL导入导出.sql文件

(1)将数据库mydb导出到mydb.sql文件
# mysqldump -u root -p mydb > mydb.mysql
(2)将数据库mydb中的表mytable导出到mytable.sql文件中
# mysqldump -u root -p mydb mytable > mytable.sql
(3)将数据库mydb的结构导出到mydb_stru.sql文件中
# mysqldump -u root -p mydb –add-drop-table > mydb.mysql
(4)将mydb.mysql文件导入到数据库中(首先mydb需要存在)
#mysql -h localhost -u root -p mydb < e:\mysql\mydb.sql
